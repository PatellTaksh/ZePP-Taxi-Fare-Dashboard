> # **ZePP – Taxi Fare Dashboard**

## Overview
The ZePP Taxi Fare Dashboard project analyzes taxi ride data to identify patterns in fare pricing, trip distance, and ride demand. The goal of this project is to transform raw ride data into meaningful insights using data analysis and visualization tools.

## Dataset
The dataset contains **15,000+ taxi ride records** across multiple attributes such as trip distance, fare amount, pickup time, and passenger count.

## Tools & Technologies
- Excel
- Power BI

## Project Workflow
1. Data collection and preparation using Excel.
2. Data cleaning and preprocessing.
3. Exploratory Data Analysis (EDA).
4. Visualization and dashboard development in Power BI.

## Key Insights
- Identified **peak ride demand hours** during the day.
- Analyzed **fare distribution patterns** across different trip distances.
- Observed trends in **pricing behavior during busy periods**.
